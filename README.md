# 📊 Web-Censo - Postes de Mombaça 2026

Projeto desenvolvido para a atividade da Unidade 5 (Provimento de Serviços
Computacionais) Capacita-irede: focado na catalogação de postes utilizando containers Docker. (Nuvem e DevOps) 

## 🛠️ Tecnologias Utilizadas
* **Node.js**: Runtime da API.
* **PostgreSQL**: Banco de dados para armazenamento geográfico.
* **Docker & Docker Compose**: Orquestração e containerização.

## 🚀 Como Executar
1. Certifique-se de ter o Docker instalado.
2. Clone o repositório.
3. No terminal, dentro da pasta do projeto, execute:
   ```bash
   docker-compose up -d
